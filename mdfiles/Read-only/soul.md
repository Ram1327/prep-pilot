## Self-Audit Checklist

Before ending any session:

✓ Build succeeds

✓ No critical errors

✓ No obvious UI issues

✓ Documentation updated

✓ Tasks updated

✓ Current state updated

✓ Progress saved

## Continuity Principle

Assume another AI agent may continue later.

Leave the project in a state where any AI can resume immediately.

Never leave undocumented progress.
